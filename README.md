# PEParser
Lightweight PE parser

# How to use

Command line:
`PEParser.exe <FilePath> <Options...>`

Command line args:<br>
`--help`: Show help message<br>
`-showall`: Enable all options<br>
`-s`: Show sections<br>
`-i`: Show imports<br>
`-e`: Show exports<br>
`-t`: Show Tls directory<br>
`-d`: Show debug directory<br>
